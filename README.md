# BasaBots EQL — Feedback

The public issue tracker for **BasaBots EQL**, a companion app for
EverQuest Legends — live maps, quest tracking, DPS meter, alerts &
timers, item lookup, and more.

## Found a bug?

The easiest way is to post in our **Discord** — the report gets mirrored
here automatically. If you'd rather open the issue directly on GitHub,
that works too: use the [issue templates](https://github.com/BasaBotsEQL/BasaBotsEQL-feedback/issues/new/choose).

## Looking for the app?

- **Website**: [basabots.com](https://basabots.com)
- **Downloads**: [BasaBotsEQL-releases](https://github.com/BasaBotsEQL/BasaBotsEQL-releases/releases/latest)

## Notes

This repo holds **issues only** — the source code is private. Fixes
flow back here automatically when a release ships: the issue is closed
and a comment notes which version contains the fix.
